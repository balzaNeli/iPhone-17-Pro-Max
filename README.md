# Site do iPhone 17 Pro

Este projeto é uma recriação da landing page do novíssimo iPhone 17 Pro Max, desenvolvido do zero. O projeto foi construído para praticar e aprimorar conceitos modernos de desenvolvimento web.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes ferramentas e tecnologias:

* **React:** Biblioteca JavaScript para construção da interface de usuário.
* **Tailwind CSS:** Framework de CSS utilitário para estilização rápida e responsiva.
* **Vite:** Ferramenta de build super rápida para iniciar projetos web.
* **Node.js & npm:** Ambiente de execução e gerenciador de pacotes necessários para as dependências do projeto.
